# Mini CRM Automation System (Python)

## Overview
This project simulates a simple CRM automation system built using Python.

It allows users to:
- add leads
- assign status automatically
- store data in a CSV file
- view stored leads

---

## Features
- Lead capture system
- Auto status assignment
- CSV-based storage
- Simple CLI interface
- Simulated email notification

---

## Tech Used
- Python
- CSV handling
- Basic automation logic

Built to practice:
